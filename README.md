**EXPLORATION OF KALMAN FILTERING IN MARINE ENVIRONMENTS**  
This project explores the effects of varying sea states on the effectiveness of Kalman filtering for state estimation. Data used is the SKLOE dataset, 
contained in this github repository https://github.com/v1ctorpan/SKLOE-dataset  
Velocities contains data pulled from the trial SKLOE dataset, with High seas as 100y, Medium as 10y, and medium as 1y conditions. Filters are located in the primary folder, 
and can be configured to pull from all velocity data. Velocity data includes derived accelerations and noise injected variants.  
This repository is paired with Report.tex as the code and data used in this project.
